# CLI

The Auth.js CLI is designed to get you up and running with Auth.js v5 in your framework of choice as quickly as possible. It is a simple, interactive command line tool that will ask you a few questions about your project and then generate the necessary files and configuration to get you started.

## Commands

### `ask`

Ask about docs, API, or auth concepts.

### `init`

Initialize a new Auth.js project.

### `add`

### `secret`

Generate a random secret for your `AUTH_SECRET` environment variable.
